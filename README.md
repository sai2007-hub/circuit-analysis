# Circuit Analysis

## Overview
This repository contains electrical and electronic circuit analysis problems, calculations, simulations, and experiments developed for engineering learning and practice.

The repository focuses on:
- Network theory
- Circuit calculations
- Current and voltage analysis
- Basic electronics concepts
- Practical circuit understanding

---

## Topics Included

### 1. Current Division Rule
Calculations and examples for parallel resistor circuits.

### 2. Voltage Division Rule
Series circuit voltage analysis problems.

### 3. Kirchhoff’s Laws
KCL and KVL circuit-solving examples.

### 4. Resistor Networks
Series and parallel resistor combinations.

### 5. Nodal Analysis
Basic node voltage calculation methods.

### 6. Circuit Simulations
Tinkercad-based electronic circuit simulations.
